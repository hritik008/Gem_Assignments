1. rem-

Font-size gets set according to the font-size of the root element.In general the html is the root element. In rem "r" stands for root.

2. em-

Font-sizes are inherited relative to the parent element. 10em means ten times of the parent element.

3. vh-

It stands for viewport height. vh is equal to the 1/100 times of the viewport height. Example: Suppose height of the browser is 1000px, so 1vh is equaled to (1/100)*1000= 10px.

4. vw-

It stands for viewport width. Similar to vh, vw is equal to the 1/100 times of the viewport width. Example: Suppose width of the browser is 1000px, so 1vw is equaled to (1/100)*1000= 10px.
